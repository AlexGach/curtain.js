  1. Fixed Sliding Panels

    a jQuery plugin by Victor Coulon

    This plugin allows you to create a web page with multiple fixed panels that unroll with an amusing effect. You can use the keyboard instead the scroll to navigate.

    I made this plugin for the new website of my friend Etienne. That's why all the illustrations on this page come from him.
    And as youp probably noticed, the effect is extensively inspired by the awesome website Edits Quarterly.

    Feel free to fork the project on github or ping me on twitter for any comments. Please be lenient. I'm currently working on the plugin.
